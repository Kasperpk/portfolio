The world *algorithm* is the term for a set of step-by-step instructions. Perhaps more formally an algorithm is a well-defined procedure that takes some  value, or set of values as *input* and produces some *output*. The word algorithm is typically linked to **computational problems**, as comptuers are machines that run programs and programs are sequences of instructions. Programs are sequences of operations falling into the definition of an algorithm and these can consist of many or few steps depending on the complexity and size of the problem.  It's an important part of computer science as all applied computer science use these concepts such as when designing hardware and software. Computer engineers pieces together billions of small devices which are capable of perform a collection of primitive operations which are algorithms. Sofware engineers then use programming languages to develop their own abstract algorithms which tackle many problems such as cryptography, database designs, web search and so on. Alot of innovation has come out of software engineering such as Google's page rank algorithm, database B-trees and so on. Applied computer science have also been able to solve complex business challenges such as finding fastest route between two points by using computational algorithms. Logistic companies use these to route their carriers as computer scientist found out that this problem could be expressed using a graph on which many clever algorithms can work on. Let's get some very basic things down for which i will refer to the book *Introduction to algorithms* by Cormen et al. They describe very basically but important that an algorithm is correct if "for every problem instance provided as input, it halts - finishes its computing in finite time - and outputs the correct solution to the problem instance" (p. 29). This is not to be forgotten as we go on to study different data structures and algorithms concepts. 

The authors also introduce some common problems solved by algorithms such are *sorting* and *searching* for which there are many practical applications like searching the fastests route between two locations. Through tackling many real world applications of programming and with the advancement of computing capacity computer scientist have developed rich set of efficient solutions for a widening array of problems like searching, sorting, various optimization problems and statistical exercises.  

While algorithms are typically linked to computational problems, if we look close enough these are everywhere arounds us. Making a cup of tea requires certain step-by-step actions and essentially all behavior whether it's consciously understood as one or not. Algorithmic thinking is like systems thinking a novel lens which the world can be viewed through and when we begin to see this we can improve the design physical and digital systems. The final motivation i will mention is that studying and solving algortihms is a great challenge that improves our general problem solving skills and thinking and it's fun to solve problems.
